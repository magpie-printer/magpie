# Extrusion Mounted Electronics

## Summary
Mount electronics to the rear extrusion of the Magpie so you don't have to laser cut acrylic.

Made by: SpiteDriven
Links: https://github.com/owenwvaughn
Version: _No response_

## Details


You'll need: 
m3 hex nuts
m3 T-Nuts
A drill
